6. **Notification**

7. **Report**

8. **Role** 

    - **Attributes:** Role_ID (PK), Role_Name, Created_At, Updated_At, Description

9. **Permission** 

    - **Attributes:** Permission_ID (PK), Permission_Name, Created_At, Updated_At, Description
